Each node is a document. Click on one to see its information. They are clustered by topic and linked if they share citations.

To filter by topic, click the checkbox on the lefthand side to hide a topic. If the node still has a topic that is visible, it will still appear. There is also the option to filter by year.

If you want to add a new document, click the 'Add source' button and enter all of the required information. The ID is a SSH ID that you get from the github repository, as long as you have permission.
